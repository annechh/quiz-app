# Quizzilly

En enkel quiz-app bygget med React, TypeScript og Vite.

## Klone prosjektet

```bash
git clone https://github.com/annechh/quiz-app.git
cd quiz-app
```

Eller open with GitHub Desktop

## Hva appen gjør

- Velg mellom flere quiz-kategorier
- Svar på spørsmål ett og ett med progresjonsbar
- Se resultat når quizen er ferdig
- Lagrer tidligere forsøk i localStorage
- Visning av svar og resultathistorikk per quiz

## Kjør prosjektet

```bash
npm install
npm run dev
```

Appen kjører som standard på Vite sin lokale dev-server.

## Nyttige scripts

- `npm run dev` starter utviklingsserver
- `npm run build` bygger produksjonsversjon
- `npm run preview` preview av build lokalt
- `npm run lint` kjører ESLint

## Teknologi

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## English

A simple quiz app built with React, TypeScript, and Vite.

## Clone the project

```bash
git clone https://github.com/annechh/quiz-app.git
cd quiz-app
```

Or open it with GitHub Desktop.

## What the app does

- Choose between multiple quiz categories
- Answer questions one by one with a progress bar
- See your result when the quiz is finished
- Saves previous attempts in localStorage
- View answers and result history per quiz

## Run the project

```bash
npm install
npm run dev
```

By default, the app runs on Vite's local dev server.

## Useful scripts

- `npm run dev` starts the development server
- `npm run build` builds for production
- `npm run preview` previews the production build locally
- `npm run lint` runs ESLint

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
